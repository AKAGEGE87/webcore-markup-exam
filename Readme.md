   ✔ Active state არსებობს

Acceptance Criteria

1. Placement
   სექცია მდებარეობს Filter Chips row-ის ქვემოთ.
   სექცია არის Main Content ზონაში.
2. Section Header
   სექციას აქვს სათაური: “Made For You”
   სათაური არის heading ტეგით (h2 რეკომენდებულია)
   სათაური ვიზუალურად გამოკვეთილია (უფრო დიდი ზომა/სქელი წონა)
3. Cards Row - Quantity
   სექციაში არის მინიმუმ 8 card ერთ რიგში (როგორც დიზაინში ჩანს).
   card-ები განლაგებულია ჰორიზონტალურად.
4. Card Structure (Semantic / DOM)
   თითო card შეიცავს მინიმუმ:
   cover image
   title (heading ან strong)
   subtitle/description (თუ დიზაინში ჩანს)
   Card უნდა იყოს ერთიანი clickable ბლოკი (a ან wrapper + link).

5. Layout Requirements (Desktop)
   Cards განლაგებულია:
   display: flex row ან grid columns გამოყენებით
   Spacing card-ებს შორის არის თანაბარი.
   Card ზომები ვიზუალურად ერთნაირია.
   თუ ეკრანზე არ ეტევა - დასაშვებია:
   horizontal scroll (overflow-x: auto) ან
   cards-ის დაპატარავება ისე, რომ ვიზუალი არ გაფუჭდეს
6. Visual Validation
   Card background/hover state ამ ეტაპზე სავალდებულო არაა, მაგრამ:
   card უნდა გამოიყურებოდეს როგორც ცალკე ბლოკი (background ან border separation)
   ტექსტები იკითხება dark background-ზე.
   Out of Scope
   card hover animations
   რეალური play button overlay ფუნქციონალი
   მონაცემების დინამიურად ჩატვირთვა
   responsive გადალაგება
   Measurable Outcome
   User Story შესრულებულად ჩაითვლება თუ:
   ✔ “Made For You” სათაური ჩანს Main Content-ში
   ✔ ქვემოთ არის მინ. 8 card ჰორიზონტალურ რიგში
   ✔ თითო card-ში ჩანს სურათი + title (+subtitle თუ საჭიროა)
